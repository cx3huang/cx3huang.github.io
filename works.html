CAD Projects |
Model – Greek Pantheon October, 2019.

A model of the Greek Pantheon done in SolidWorks, originally for a school project. Used typical modelling tools in SolidWorks, and the PhotoView360 Add-in to edit surface materials and produce a render.

Assembly – Bus November, 2019.

A model of a bus done in SolidWorks. This assembly includes the side mirror parts, the bus part, and the bike holder part. For this model, I explored using assemblies.

Model – Plane November, 2019.

A plane model done in SolidWorks, also originally for a school project. For this model, I mainly focused on creating a lofted, smooth shape with various modeling tools.

Animation and Rendering Projects |
Exterior Lighting in 3ds Max april, 2020.
❮ ❯

These renders were done in 3ds Max to experiment with different lighting tools. Some used for these include skydome with HDRI images and sunlight positioner. The model was done in 3ds Max as well. This was an interesting project to explore the Arnold renderer specifically!

Maskora November, 2020.
❮ ❯

A project that I did with some friends at the UofT Designathon 2021. This design was prompted by a need to prevent glasses from fogging up while wearing a mask. We used Fusion 360 and SolidWorks to model and 3ds Max to render the GIF.

UX/UI & Software Projects |

At StarterHacks, my team created an algorithm that connects donating organizations to women's shelters in need. We used C# for the algorithm and JavaScript to make an online platform.
